What are Buffer Overflows?

Buffer overflow vulnerability affects the web applications that require user input. When the input that is sent to the application is more than the buffer capacity and the buffers are left unchecked, buffer overflow occurs

What are Application Firewall?

Application firewalls analyze the requests at the application level. These firewalls are used for specific applications like a web server or a database server. The web application firewalls protect the web server from HTTP based attacks.

Why can't I trust the information coming from the browser?

There are chances that the information is modified before it reaches the server. Attackers browsing the site can manipulate the information in a GET or POST request. Although the client and the server may be trusted, we cannot be sure that the information is not modified after it leaves the browser. Attackers can capture the information on the way and manipulate it.
