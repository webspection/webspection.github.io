Automated Threat

This is a threat events to applications undertaken using automated actions. The focus is on abuse of functionality - misuse of inherent functionality and related design flaws

What is SQL injection attack?

It is an attack that allows attacker to use queries with the database. It allows attacker to view data that should not be available for retrieval


What is the danger of SQL injection attack?

Attacker can use SQL injection attack to modify or delete the data in the database and cause persistent changes to the website. In some case, attacker can even use SQL injection attacker to compromise the server or perform further attack such as DoS attack
