Automated Threat

This is a threat events to applications undertaken using automated actions. The focus is on abuse of functionality - misuse of inherent functionality and related design flaws

What is UDP flood attack?

UDP flood is a type of DDoS attack that a large amount of UDP packets are sent to the targeted server to overwhelming the service to respond to legitimate users


What is SYN flood attack?

SYN flood is a type of DDoS attack what attacker uses TCP protocol to flood a system with repeated SYN request to make a system unresponsive to normal users


What is ping of death attack?

Ping of death is a type of DDoS attack that attacker try to crash the targeted server by sending malformed or oversized packet with a ping command


What is smurf attack?

Smurf attack is a type of DDoS attack that attacker try to send a large amount of ICMP request packets to the targeted server. Smurf is an amplification attack that uses exploit of broadcast networks
